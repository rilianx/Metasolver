Software Installation Guide

Before using the Gurobi Optimizer, you'll need to install the software on your computer. If you haven't already done so, please go to our download page. Find your platform (we'll assume Mac OS X in this document) and choose the corresponding file to download.

Make a note of the name and location of the downloaded file.

Your next step is to double-click on the appropriate Gurobi installer (e.g., gurobi8.1.0_mac64.pkg for Gurobi 8.1.0) and follow the prompts. By default, the installer will place the Gurobi 8.1.0 files in /Library/gurobi810/mac64 (note that this is the system /Library directory, not your personal ~/Library directory). Your <installdir> (which we'll refer to throughout this document) will be /Library/gurobi810/mac64.

You are now ready to proceed to the section on Retrieving Your Gurobi License.

If you would like an overview of the files included in the Gurobi distribution, you can also view the File Overview section.
